Our road map helps us share our priorities. If React observing currently doesn't have a
specific feature you think it's cool, [check
it](https://github.com/lvsouza/react-observing/issues)
to see if it's planned for the future. Otherwise, we recommend you
[create an issue](https://github.com/lvsouza/react-observing/issues) describing your enhancement request.
